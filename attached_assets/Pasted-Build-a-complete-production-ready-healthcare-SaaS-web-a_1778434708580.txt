Build a complete production-ready healthcare SaaS web application called SMART CLINIC.

SMART CLINIC is a modern multi-clinic healthcare management and queue management platform designed for clinics, healthcare staff, administrators, and patients.

The system must be scalable, secure, responsive, modern, and designed for real-world clinic operations.

==================================================
MAIN PURPOSE OF THE SYSTEM
==========================

SMART CLINIC helps clinics:

* manage patients
* manage queues
* manage appointments
* manage clinic staff
* reduce waiting times
* digitize clinic operations
* improve clinic workflow
* support future patient mobile apps

The platform must support multiple independent clinics using the same system.

Each clinic must have its own:

* patients
* staff
* appointments
* queues
* analytics
* records

No clinic should ever access another clinic’s data.

==================================================
IMPORTANT SYSTEM OPERATION
==========================

The platform must work like this:

STEP 1:
A clinic admin creates a new clinic.

STEP 2:
The system generates a unique clinic code.

Example:
SC-582104

STEP 3:
The clinic admin shares this clinic code with clinic staff.

STEP 4:
Staff members create accounts.

STEP 5:
Instead of creating clinics, staff users click:
"Join Existing Clinic"

STEP 6:
Staff enters:

* clinic code
* role request

STEP 7:
The clinic admin receives join requests.

STEP 8:
Clinic admin approves or rejects users.

STEP 9:
Approved users become members of the SAME clinic.
